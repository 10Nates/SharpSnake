# SharpSnake

A terminal snake game written in C#. 

Comes with a rudimentary pathfinding algorithm designed by me.

## Controls:
| Key | Action |
|---|---|
| W/A/S/D | control movement |
| Tab | Show debug info |
| P | Activate Pathfinding |
| L | Run as fast as possible |
| K | Kill snake |
| Esc / C | Close |
